# input-patterns

> 

## Install

```
$ npm install input-patterns
```

## Usage

```jsx
const pattern = require('input-patterns');

// In React, for example
<input type="text" pattern={`${pattern.alphaNumeric}`} title="Please use only alpha-numeric characters">
```
